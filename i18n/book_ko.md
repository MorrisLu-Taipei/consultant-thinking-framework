# 컨설팅 사고력 시스템
*컨설팅 사고를 육성하고 평가하기 위한 운용 가능한 프레임워크*

저자: Morris (Yeh-Hsing) Lu · LinkedIn · ORCID 0009-0006-5373-0586 · Apache License 2.0

> **북극성(지침).** 고객이 "무엇을 말했는지" 기억하길 바라는 게 아니다. 고객이 **진짜로 무엇을 필요로 하는지**, 그리고 **다음 수로 어떻게 이길지**를 판단하길 바란다.

**정직한 포지셔닝.** 이것은 새로운 전략 프레임워크가 아니다. 컨설팅 사고를 **훈련 가능·채점 가능·검증 가능**하게 만드는 시스템이다. 오마에의 3C, TOWS, Temporal Competitive Analysis, 카노 모델 위에 서 있다. 고유한 기여는 (a) "주체 × 시간"을 하나의 직교 행렬에 놓은 것, (b) 선택적 "대각선" 종합, 두 가지다. **아직 실증되지 않았다.**

## 1. 행렬
네 가지 렌즈 — **고객·경쟁사·자사·환경** — × 세 시점: **과거(원인)·현재(needs)·미래(해법)**. 각 칸을 구분한다:
- 📥 **고객 입력** = 듣게 되는 원재료(주어지는 것).
- 🧠 **컨설턴트 산출** = 스스로 만들어내는 것(가치는 여기 있다).

📥(입력)은 고객의 과거·현재의 "표면"뿐이다. 나머지 칸은 대부분 🧠(산출)이다. C1인 사람은 📥 칸만 채운다(고객 발언의 복사). 가치는 🧠 칸 **+ 대각선**에 전부 있다.

## 2. 세 가지 읽기
- **세로 = 벡터**: "어떤 힘이 과거를 현재로 밀었는가"에서 미래를 **추론**한다. 짐작하지 않는다.
- **가로 = 격차**: 고객/경쟁사/자사/환경의 비대칭이 기회다.
- **대각선 = 보수의 원천**: 주체 × 시간을 가로질러 잇는다. 예: *경쟁사의 "현재" 맹점 × 고객의 "미래" 니즈 × 환경의 "과거" 구조 변화 → 제안*.
> 칸을 채우는 것은 사고가 아니다. 가치는 대각선에 있다.

## 3. 역량 사다리
C1 기록 → C2 정리 → C3 분석 → C4 기획(대각선 하나) → C5 자문(다수의 대각선 + 말하지 않은 니즈 + 자사/환경). 대부분은 C1에 머문다.

## 4. 컨설팅 요약(6부)
1. 고객의 원문(사실). 2. 진짜 통점(표면 → 근본). 3. 현재 니즈(해결하려는 과업). 4. 미래 제안 **+ 말하지 않은 니즈**. 5. 경쟁과 환경. 6. 우리가 들어가는 방법(해결 가능/불가, 어느 표준 제품). 더하여: **다음 단계**와 **명시적 대각선 최소 1개**.

## 5. 제출 전 규칙
"**경쟁사는 현재…, 게다가 환경 트렌드…, 따라서 고객은 미래…**" 같은 문장이 하나도 없다면 아직 제출할 수 없다. 채점에서: **대각선이 없으면 C3 상한. 물어본 것만 답하면 최대 C4.**

## 6. AI와 사용하기
프롬프트를 외울 필요 없다. "이 회의 메모를 컨설팅 사고 행렬로 첨삭해줘" 또는 "루브릭으로 채점해줘"라고 말하면 된다. 규율: **신호와 추론을 구분**하고, 날조하지 말며, 미확인은 **TBD**로 표기한다.
