# ระบบสมรรถนะการคิดเชิงที่ปรึกษา
*กรอบที่นำไปปฏิบัติได้ เพื่อพัฒนาและประเมินการคิดเชิงที่ปรึกษา*

ผู้เขียน: Morris (Yeh-Hsing) Lu · LinkedIn · ORCID 0009-0006-5373-0586 · Apache License 2.0

> **เข็มทิศ.** ผมไม่ได้อยากให้คุณจำว่าลูกค้าพูดอะไร แต่อยากให้คุณ**ตัดสินว่าลูกค้าต้องการอะไรจริง ๆ** และเรา**จะชนะในก้าวต่อไปอย่างไร**

**การวางตำแหน่งอย่างตรงไปตรงมา** นี่ไม่ใช่กรอบกลยุทธ์ใหม่ แต่เป็น**ระบบที่ทำให้การคิดเชิงที่ปรึกษา ฝึกได้ ให้คะแนนได้ และตรวจสอบได้** ตั้งอยู่บนไหล่ของ 3C ของโอมาเอะ, TOWS, Temporal Competitive Analysis และโมเดล Kano จุดที่เพิ่มเข้ามาคือ (ก) วาง «ผู้เกี่ยวข้อง × เวลา» ลงในตารางตั้งฉากเดียว และ (ข) การสังเคราะห์แบบ «เส้นทแยง» เชิงเลือก — **และยังไม่ได้รับการพิสูจน์เชิงประจักษ์**

## 1. เมทริกซ์
สี่มุมมอง — **ลูกค้า · คู่แข่ง · เรา(บริษัท) · สภาพแวดล้อม** — ตามสามช่วงเวลา: **อดีต(สาเหตุ) · ปัจจุบัน(ความต้องการ) · อนาคต(ทางออก)** ทำเครื่องหมายแต่ละช่อง:
- 📥 **อินพุตจากลูกค้า** = วัตถุดิบที่คุณได้ยิน (สิ่งที่ถูกให้มา)
- 🧠 **เอาต์พุตของที่ปรึกษา** = สิ่งที่คุณสร้างขึ้น (คุณค่าอยู่ตรงนี้)

มีเพียง «ผิวหน้า» ของอดีต/ปัจจุบันของลูกค้าที่เป็นอินพุต 📥 ส่วนช่องที่เหลือส่วนใหญ่เป็นเอาต์พุต 🧠 คนระดับ C1 เติมได้แค่ช่อง 📥 (ลอกคำลูกค้า) คุณค่าอยู่ที่ช่อง 🧠 **+ เส้นทแยง** ทั้งหมด

## 2. สามวิธีอ่าน
- **แนวตั้ง = เวกเตอร์**: อนุมานอนาคตจาก «แรงใดที่ผลักอดีตมาสู่ปัจจุบัน» อย่าเดา
- **แนวนอน = ช่องว่าง**: ความไม่สมมาตรระหว่างลูกค้า/คู่แข่ง/เรา/สภาพแวดล้อมคือโอกาส
- **เส้นทแยง = ค่าที่ปรึกษา**: เชื่อมข้ามผู้เกี่ยวข้อง × เวลา เช่น *จุดบอด「ปัจจุบัน」ของคู่แข่ง × ความต้องการ「อนาคต」ของลูกค้า × การเปลี่ยนเชิงโครงสร้าง「อดีต」ของสภาพแวดล้อม → ข้อเสนอ*
> การเติมช่องไม่ใช่การคิด คุณค่าอยู่ที่เส้นทแยง

## 3. บันไดสมรรถนะ
C1 บันทึก → C2 จัดระเบียบ → C3 วิเคราะห์ → C4 วางแผน(เส้นทแยงหนึ่งเส้น) → C5 ให้คำปรึกษา(หลายเส้นทแยง + ความต้องการที่ไม่ได้พูด + เรา/สภาพแวดล้อม) คนส่วนใหญ่ติดอยู่ที่ C1

## 4. สรุปการประชุมเชิงที่ปรึกษา (หกส่วน)
1. คำพูดลูกค้าตามจริง(ข้อเท็จจริง) 2. ความเจ็บปวดที่แท้จริง(ผิวหน้า → ราก) 3. ความต้องการปัจจุบัน(งานที่อยากให้สำเร็จ) 4. ข้อเสนออนาคต **+ ความต้องการที่ไม่ได้พูด** 5. คู่แข่งและสภาพแวดล้อม 6. เราจะเข้าอย่างไร(แก้ได้/แก้ไม่ได้; ผลิตภัณฑ์มาตรฐานตัวไหน) เพิ่มเติม: **ก้าวต่อไป** และ **เส้นทแยงชัดเจนอย่างน้อยหนึ่งเส้น**

## 5. กฎก่อนส่ง
หากรายงานของคุณไม่มีประโยคทำนอง «**เพราะคู่แข่งตอนนี้… บวกกับเทรนด์สภาพแวดล้อม… ลูกค้าในอนาคตจะ…**» แสดงว่ายังไม่พร้อมส่ง ในการให้คะแนน: **ไม่มีเส้นทแยง → สูงสุด C3; ตอบเฉพาะที่ถูกถาม → สูงสุด C4**

## 6. ใช้กับ AI
ไม่ต้องท่องพรอมต์ เพียงพูดว่า «ช่วยตรวจบันทึกประชุมนี้ด้วยเมทริกซ์การคิดเชิงที่ปรึกษา» หรือ «ให้คะแนนด้วยรูบริก» วินัย: **แยกสัญญาณกับการอนุมาน** อย่ากุขึ้นเอง สิ่งที่ยังไม่ยืนยันให้ทำเครื่องหมาย **รอยืนยัน**
