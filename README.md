## 🧭 核心(一頁版)
- **矩陣**:客戶 / 對手 / 我們 + 環境 × 過去 / 現在 / 未來;分清 📥 客戶輸入 vs 🧠 顧問產出。
- **三讀法**:縱向=向量、橫向=落差、**對角線=顧問費**(跨主體×跨時態連點)。
- **能力梯**:C1 記錄 → C2 整理 → C3 分析 → C4 規劃(一條對角線)→ C5 顧問(多對角線+沒說出口+我們/環境)。
- **防呆**:無對角線封頂 C3;只回答客戶問的最多 C4。
