# AI Prompt Pack(讓 AI 幫你跑這套方法)

> 用法:複製貼到 AI(或裝了 `consulting-thinking-coach` skill 後直接講白話)。每個 prompt 都附「貼什麼進去」。
> 紀律:要求 AI **區分訊號 vs 推測、不編造**;沒證據的標「待確認」。

---

## P1 · 逐字稿/紀錄 → 顧問式摘要
```
你是顧問思考矩陣教練。把下面的會議內容改寫成「顧問式會議摘要」,六段:
1 客戶原話(事實) 2 真正痛點(表面→真因) 3 現在需求(用 JTBD:他想完成什麼)
4 未來建議＋「沒說出口」的潛在需求 5 競爭與環境(對手盲點 + 法規/趨勢) 6 我們怎麼切入(能解/不能解/怎麼賺/怎麼產品化)
最後:7 下一步。並「明寫至少一條對角線」(對手或環境的某時態 × 客戶某時態),說明為什麼成立。
區分「客戶真訊號」與「你的推測」,沒證據標「待確認」,不要編造。
【貼會議逐字稿/紀錄】:
```

## P2 · 用 Rubric 評分
```
用「顧問能力評估表」幫這份摘要打分:六項(紀錄/痛點/未來/競爭/市場環境/商業判斷)各 0-2,給分+理由。
套規則:沒有任何成立的對角線→封頂 C3;只回答客戶問的→最多 C4。
算總分(12)、判 C1-C5、指出「最該補的一格」與「升一級的下一步」。
【貼摘要】:
```

## P3 · 找缺的對角線
```
對照 3 主體(客戶/對手/我們)+環境 × 3 時態矩陣,指出這份摘要:哪些格是空的、哪條對角線沒連起來。
提 1-2 條「可成立的對角線候選」,但只當假設讓我驗證,別替我斷定沒證據的事。
【貼摘要或紀錄】:
```

## P4 · 主管式反問(帶人用)
```
我是主管,部下交來這份(偏 C1 紀錄)。用 Socratic 反問,別罵,生 4 個問題分別逼他想:未來向量 / 對手盲點 / 環境跨城跨國 / 我們能不能解。最後一個問題逼他把三點連成一條對角線。
【貼部下的紀錄】:
```

## P5 · 一句話品質檢查
```
這份摘要有沒有「判斷」,還是只有「逐字稿」?用一句話回答 C 幾級,並指出有沒有對角線、有沒有抓到沒說出口的需求。
【貼摘要】:
```

---
## 最短用法(裝了 skill 後)
直接講:**「用顧問思考矩陣批改這份會議紀錄」** / **「幫我用 rubric 評分」** / **「這份缺哪條對角線」**。
